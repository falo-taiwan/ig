1. v2.1 重點功能升級邏輯

A. 多國語系切換器 (i18n Switcher)

導覽列右上角的國旗按鈕點擊後展開下拉選單，點選即可在 🇹🇼 繁體中文、🇺🇸 English、🇹🇭 ภาษาไทย、🇻🇳 Tiếng Việt 間即時進行無縫切換。所有主菜單、頁籤、購物車、驗收提示及後台表格均支援全語系翻譯。

B. WooCommerce 商品客製加選彈窗 (Options Customizer)

點擊商品卡片時不再直接加入購物車，而是彈出加選視窗。使用者可選擇米飯份量（正常、加量 +$10、減半）、客製化要求（不要油蔥、加生辣椒、免洗餐具）或會議餐盒預算（$200/$250）。購物車會將這些客製化項目打包（如 options: "米飯加量,不要油蔥"）並即時累加價格。

C. 直達結帳滾動 (Direct Checkout)

點選 Navbar 的「結帳」時，系統會自動呼叫 openCheckoutDirectly()。當檢測到購物車有商品時，會自動展開購物車並以平滑動畫 (Smooth Scroll) 捲動至下方「配送明細」表單，提升結帳流暢度。

D. 基本資料與送餐地址簿 (Member Address Book)

會員可在個人專區設定姓名、電話、公司與常用外送地址。設定儲存後，LocalStorage 會將資料快取，並在結帳時自動預填 (Autofill) 配送表單。

2. 鼓勵與反饋留言板系統

為高仿原站 slk9898.com.tw/鼓勵與反饋/，我們為訪客與管理員實現了完整的留言功能：

  • 訪客/會員模式：提供乾淨的留言表單。留言時必須輸入系統中合法的「員工工號」（如 1000、11100002 到 11100020），否則會彈出工號錯誤警告。驗證成功後，系統會自動在人員庫中匹配 mock 姓名與部門，並記錄當下時間。
  • 管理員(後台)模式：當使用者在底部切換為 Admin 權限時，此分頁會自動渲染成留言管理後台表格：
    • 即時篩選：可輸入工號進行模糊查詢，並透過下拉選單過濾「已讀」或「未讀」留言。
    • 批量修改：勾選多筆留言後，可點選「標示為已讀」或「標示為未讀」進行批量狀態變更。
    • 行內編輯與刪除：點擊修改圖示即可彈出 prompt 編輯留言內容，點選刪除圖示並確認後即可將其完全刪除。

3. 獨立的 GAS 部署指南面版

原本混雜在回饋分頁中的 Google Apps Script (GAS) 指南與代碼展示器，已完全被抽離到 Navbar 新增的 「GAS 部署」 獨立分頁中，並套用紫色系主題。代碼預覽框提供 Code.gs v2.1 完整後端原始碼，並提供「複製代碼」功能，方便學員一鍵部署。
