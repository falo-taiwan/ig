# FALO Instagram Video Enhancer (v1.0.1)

> **Falo x Force Cheng 2026/6/8**

本專案是一個 Instagram 網頁版影片控制 Chrome 外掛程式。以下是手動載入與安裝至 Chrome 瀏覽器的步驟。

---

## 🛠️ Chrome 外掛程式 本地安裝步驟

1. **下載原始碼**：下載本專案中的 `FALO_Instagram_Video_Enhancer.zip` 並在您的電腦上進行解壓縮。
2. **進入擴充功能管理頁**：開啟 Google Chrome 瀏覽器，在網址列中輸入 `chrome://extensions/` 並按下 Enter 鍵。
3. **開啟開發人員模式**：在擴充功能管理頁面的右上方，找到**「開發人員模式」(Developer mode)** 開關，並將其切換為**啟用**狀態。
4. **載入未封裝項目**：點擊頁面左上方浮現的 **「載入未封裝項目」(Load unpacked)** 按鈕。
5. **選擇外掛資料夾**：在彈出的視窗中，選取解壓後得到的 `ig-video-enhancer-extension` 資料夾，點選確認。
6. **完成安裝**：當畫面上出現 **FALO Instagram Video Enhancer v1.0.1** 記憶卡片且狀態開關已啟用，即表示外掛已順利在本地部署完成！
